# Hid.js

Hid interpreter implemented in javascript.

## Examples

```

const Hid = require('Hid');

Hid.run("print \"hello, world!\"");

```

## HidTranspiler.js

HidTranspiler.js uses the AST generator to generate pretty JS code (0.0.1)
